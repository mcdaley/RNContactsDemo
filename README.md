# RNavigationDemo
An example React Native app that demonstrates how to use React Navigation
to setup the layout for an app. The demo will show:

* Tabbed Navigation
* Stacked Navigation
* Drawer Navigation
* Modal Windows
* Header Styling
  - Text
  - Buttons/Icons/Images
  - Search Window
* Bottom Tab Styling
  - Text
  - Images/Icons
* Authentication Flow
  - Sign Up
  - Sign In
  - Forgot Password
